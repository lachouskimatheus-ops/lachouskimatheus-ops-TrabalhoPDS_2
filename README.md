Esse projeto consiste no trabalho de PDS-2 dos integrantes: Matheus Lachouski, Igor Sassano, João Luís, Cássio J, Emanuel Costa e Lucas Almeida.
O trabalho envolve a criação de uma central de jogos que usam o clássico baralho como base, sendo eles: Truco, Pife, Paciência, Rouba Monte, Poker e Fodinha.
