Este projeto foi desenvolvido como trabalho da disciplina de PDS-2, com o objetivo de implementar uma central de jogos clássicos baseados em baralho, utilizando C++ e conceitos de modularização e reaproveitamento de código.
A proposta central é construir uma arquitetura reutilizável, onde componentes fundamentais como carta e baralho possam ser compartilhados entre diferentes jogos, garantindo organização, escalabilidade e manutenção simplificada.
A central terá os seguintes jogos: Truco, Pife, Paciência, Rouba Monte, Poker, Fodinha
